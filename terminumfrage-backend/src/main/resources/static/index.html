Terminumfrage Backend

Dein Spring Boot Server läuft. Von hier aus erreichst du sowohl die REST-API als auch die Angular-Oberfläche (via Proxy).

Los geht’s

  • API-Health: curl http://localhost:8080/actuator/health (falls Actuator aktiviert ist).
  • H2-Konsole: /h2-console (JDBC URL jdbc:h2:mem:terminumfrage).
  • Poll-Endpunkte: Authentifiziert erreichbar unter /api/v1/polls.

Angular-Frontend starten

  1. cd terminumfrage-frontend
  2. npm install (erstinstallation)
  3. npm start oder ng serve --proxy-config proxy.conf.json

Die Proxy-Konfiguration leitet Aufrufe von /api automatisch an dieses Backend weiter.

Angular-Frontend öffnen API ausprobieren (Login nötig)
Statusseite automatisch bereitgestellt von Spring Boot – passe den Inhalt unter src/main/resources/static/index.html nach Bedarf an.
